# @amaneku/eslint-config

## Installation

Install `@amaneku/eslint-config`

```
$ npm install --save-dev @amaneku/eslint-config
```

## Usage

in `.eslintrc.js`

```js
module.exports = {
  extends: ['@amaneku/eslint-config']
};
```

es5:

```js
module.exports = {
  extends: ['@amaneku/eslint-config/es5']
};
```
